# Movie Directory React Application

This product uses the TMDB API but is not endorsed or certified by TMDB.

This is made using Node.js, React, CSS, and HTML. In order to start this project, you must
download the required installs by navigating to where the file package.json is and typing

    npm install

in your terminal.

To run the project, you must type

    npm run dev

and this should show up on your web browser. If it does not show up automatically,
*ctrl+click* the url.

This app shows all of the currently popular movies on the front page. You are able to
scroll through all of the available ones. If you do not see a movie that you are looking
for, you may use the search bar to find it. You may favorite/unfavorite any of these 
movies. They will then show up in the ***Favorites*** tab in the top righthand corner.
